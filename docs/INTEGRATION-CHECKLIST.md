# Integration Checklist

## Purpose
Define what each integration must support for Phase 1 demos.

## Meetings
### Microsoft Teams
- Meeting metadata capture
- Transcript ingestion
- Participant list

### Google Meet
- Meeting metadata capture
- Transcript ingestion
- Participant list

### Zoom
- Meeting metadata capture
- Transcript ingestion
- Participant list

## CRM
### Salesforce
- Create/update tasks
- Link meeting summary to account

### HubSpot
- Create/update tasks
- Link meeting summary to contact

## Work Management
### Jira
- Create tasks from action items

### Confluence
- Create summary page from meeting recap
