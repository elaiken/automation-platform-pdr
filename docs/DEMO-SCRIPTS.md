# Demo Scripts

## Purpose
Provide step-by-step talking points to deliver consistent demos across industries.

## Healthcare Demo Script
1) Open Owner Dashboard
   - Highlight care-task completion and compliance snapshot
2) Open Meeting Recap
   - Point out decisions and action items
3) Show action routing
   - Assign care coordinator and due date
4) Show document linking
   - Clinical notes attached with restricted access
5) Compliance callout
   - Audit trail entry and retention policy

## Financial Services Demo Script
1) Open Dashboard
   - Supervision SLA and recordkeeping status
2) Open Meeting Recap
   - Decisions logged with timestamps
3) Approval workflow
   - Compliance review required before sharing
4) CRM sync
   - Client follow-up created automatically
5) Audit export
   - Show export-ready audit log

## Professional Services Demo Script
1) Open Dashboard
   - Matter status and utilization
2) Open Meeting Recap
   - Matter-specific summary and actions
3) Privilege handling
   - Restricted access to sensitive notes
4) Knowledge base
   - Reusable insight stored
5) Work delivery
   - Task creation and assignment
