# Demo Flows

## Purpose
Provide clear, repeatable demo narratives that show end-to-end value for each target industry.

## Demo 1: Healthcare (Clinic Operations)
### Scenario
Weekly care coordination meeting for high-risk patients.

### Flow
1) Meeting capture starts automatically from calendar invite.
2) Live transcription + post-meeting summary (decisions + action items).
3) Action items routed to care coordinators with due dates.
4) Patient-related documents tagged and stored with restricted access.
5) Owner dashboard shows completion rate and overdue risk.

### Key Value
- Faster handoffs
- Reduced missed follow-ups
- Clear HIPAA-friendly audit trail

### Success Criteria
- 80%+ action items assigned within 10 minutes of meeting end
- 30% reduction in overdue care tasks

## Demo 2: Financial Services (Wealth Management)
### Scenario
Quarterly portfolio review with compliance requirements.

### Flow
1) Meeting summary generated with decisions and client commitments.
2) Compliance review step required before external sharing.
3) Decisions logged into audit trail automatically.
4) CRM updates created for client follow-ups.
5) Dashboard highlights regulatory risk and SLA adherence.

### Key Value
- Audit-ready records
- Reduced manual note-taking
- Faster compliance checks

### Success Criteria
- 100% of meetings logged with audit trail
- 50% reduction in time spent on compliance prep

## Demo 3: Professional Services (Legal/Consulting)
### Scenario
Client strategy workshop and follow-up planning.

### Flow
1) Meeting capture generates matter-specific summary.
2) Privileged content tagged and access-restricted.
3) Tasks created for drafting and review steps.
4) Knowledge base updated with reusable insights.
5) Owner dashboard shows team utilization and delivery risks.

### Key Value
- Matter-centric knowledge organization
- Reduced admin overhead
- Faster proposal and delivery cycles

### Success Criteria
- 25% improvement in delivery cycle time
- 90%+ on-time task completion
