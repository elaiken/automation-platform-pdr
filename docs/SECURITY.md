# Security and Compliance

## Principles
- Least privilege by default
- Clear auditability for regulated decisions
- Encryption in transit and at rest

## Access Control
- Role-based access control (RBAC)
- Client and matter segmentation
- Admin-configurable sharing rules

## Audit and Retention
- Immutable audit logs for critical actions
- Configurable retention by industry pack
- Exportable audit trails for regulators

## Data Handling
- Sensitive data classification
- DLP-style restrictions for exports
- Redaction for external sharing

## Regulatory Targets (Phase 1)
- HIPAA (Healthcare)
- SEC/FINRA recordkeeping (Financial Services)
- Privilege-aware handling (Professional Services)

## AI Risk Controls
- Human approval for regulated outputs
- Source citations for AI summaries
- Monitoring for drift and errors
